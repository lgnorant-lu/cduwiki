# 校园网移动电信联通wifi和宽带配置教程 (win11版)

## step1: 在"开始菜单"中打开设置

![windows11-开始-设置](assets/Snipaste_2024-07-11_23-40-20.png)

## step2: 依次点击“网络 和Internet”、“拨号”

![依次点击“网络 和Internet”、“拨号”](assets/图片1.png)



## step3: 点击“设置新连接”

![点击“设置新连接”](assets/图片2.png)

## step4: 选择“连接到Internet”，点击下一步

![选择“连接到Internet”，点击下一步](assets/图片3.png)

## step5: 点击“设置新的连接”

![点击“设置新的连接”](assets/image-20240711234806237.png)

## step6: 选择“宽带（PPPoE）”

![选择“宽带（PPPoE）”](assets/image-20240711234829759.png)

## step7: 填写用户名密码

 [宽带密码.md](宽带账号密码.md) 

![填写用户名密码](assets/图片4.png)

## 连接成功后关闭改窗口即可正常访问网络

![连接成功后关闭改窗口即可正常访问网络](assets/图片5.png)
