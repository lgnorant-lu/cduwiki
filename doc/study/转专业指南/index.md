# 转专业教程指南

本转专业教程指南由学院内部学长学姐撰写提供实用的建议，帮助学生顺利转入心仪的专业。无论你是对新领域充满热情，还是希望提升职业前景，我们的指南将为你提供全面的信息和有用的资源，助你实现学术和职业目标。

建议同时阅读[对应学院的内部信息](../学院内部信息/index.md), 特别是工作室. 提前与导师建立良好的关系会让你转专业复试的过程事半功倍
